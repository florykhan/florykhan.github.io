Live Demo GitHub Repo
In Progress

Problem & Context

Control systems for physical systems like the inverted pendulum often benefit from data-driven and learning-based approaches. The goal of this project is to design a data-driven control system using reinforcement learning to stabilize the pendulum, combining control theory with modern ML techniques. This is a team project currently in progress.

What We’re Building

  • Data-driven or reinforcement-learning-based controller for inverted pendulum dynamics
  • Integration of simulation and/or real hardware with the learning pipeline
  • Evaluation of stability and performance under the learned policy
